引言

在购买笔记本电脑之前，你还得了解一些电脑相关的信息，才能更加清楚应该如何选购笔记本电脑


我给你大致介绍一下电脑内部的电子部件，以及电脑相关的专有名词，这有助于你完善对于电脑的感知

CPU - 中央处理器

CPU用于处理各个软件的常规运行，它决定电脑的任务处理速度和复杂问题的解决能力。

若将一部电脑比喻成一个团结协作的团队，CPU即是团队中的决策者和领导者，负责决策团队的活动，领导者（CPU）越"聪明"，越能快速分析办公桌（内存）上的资料（文件）并安排任务（处理各软件的运行），但若领导者反应慢，团队时常会卡在开会讨论（CPU计算瓶颈）之中，而无法继续推进项目（如打开大型软件较慢、切换软件的时候有延迟），所以CPU将是等会儿选购笔记本的重要因素之一。

GPU（显卡） - 图形处理器

GPU用于渲染屏幕画面，它决定电脑的图形处理速度和画面流畅度。

沿用以上例子，GPU即是团队中的设计师，负责绘制图像（渲染屏幕画面）给用户（电脑使用者）观看，专业的设计师（高性能GPU）能更快速地绘制图像、进行3D建模，普通的设计师（低性能GPU）在绘制图像时更缓慢（如游戏时FPS较低），GPU同样是等会儿选购笔记本的重要因素之一。

内存 (RAM) - 文件临时存放处

内存是便于CPU快速访问数据的硬件，它决定电脑能同时处理多少任务以及运行多少软件。

沿用以上例子，内存是团队的办公桌，办公桌面积越大，能放在桌子上的文件资料（数据）越多，可供领导者（CPU）同时使用的文件越多（当然，也要领导者足够"聪明"，才能同时分析很多的文件），倘若桌子太小，领导者则要频繁跑到文件柜（硬盘）里翻找文件（如在开启多个软件后卡顿）。对你来说，16GB的内存刚好合适。

硬盘 - 文件长期存放处

硬盘是电脑长期存放文件的地方，用于保存你的游戏，软件等等各种各样的文件，它决定了数据载入和保存的快慢。

沿用以上例子，硬盘是团队的文件柜，专门存放资料或档案（电脑中的文件），文件柜体积越大或越小，对电脑的影响自不必多说，这决定了你能放多少东西；另一方面，若你的文件柜越智能（硬盘读写速率越高），翻找资料的速度就越快，能更快地将文件放到办公桌（内存）上，而你的文件柜越低端，翻找文件速度就越慢（如开机慢、软件启动较久、游戏加载时间较长）。

你可能不知道上一页的"FPS"是什么意思，那么接下来介绍诸如"FPS"这样的计算机专有名词


这一页不仅介绍了专有名词，同时也普及一些基本常识

刷新率与FPS

  • 刷新率：这是笔记本电脑的屏幕的其中一个参数，它决定了你的屏幕每秒最多可以显示多少张画面，单位是Hz，如144Hz即每秒最多显示144张画面
  • FPS：即帧率，该数值是软件每秒生成的画面数，它决定了你的屏幕每秒输出多少张软件画面，单位是FPS（即frame per second - 帧/秒），如120FPS，即120帧，就是每秒刷新120次画面

看上去刷新率和帧率的定义好像差不多，那么他们具体有什么区别呢？

刷新率(Hz) FPS(帧率)
本质 屏幕的硬件参数 软件的实际输出画面速率
决定因素 屏幕本身 GPU性能及软件优化
上限 固定值 理论无上限
感知效果 滚动网页、鼠标移动等操作的流畅度 游戏、视频的流畅度与连贯性

可以看出，高刷新率只是让高帧率的潜力被充分利用，实际流畅度还是由帧率决定


值得一提的是，GPU性能越强，一秒内能渲染出的画面数越多，帧率也就越高，游戏画面越流畅，反之则越卡顿


以下分别是15帧、30帧以及60帧的效果演示

像素（Pixel）

像素是图像的最小单位，可以理解为一个个小色块，每个像素显示一种颜色，多个像素组合起来便形成了完整的画面

分辨率

分辨率指的是屏幕的像素数量，通常用横向像素数 × 纵向像素数表示，例如1920*1080＝横向1920像素*纵向1080像素＝总共207万像素，以下是分辨率常见标准：

  • 720p (1280×720) - 入门级清晰度，常见于旧设备
  • 1080p (1920×1080) - 全高清，是主流电脑及电视标准
  • 2K (2560×1440) - 比1080p更细腻
  • 4K (3840×2160) - 超高清，细节丰富，适合专业设计及影视

分辨率越高，GPU在每一帧需要渲染的像素数量越多，对GPU性能要求也就越高，相同性能的GPU，在2K屏上输出的帧率比在4K屏上高得多


综上所述，屏幕分辨率的选择与GPU的选择通常要相对应，一般不会拿一个性能较低的GPU去搭配4K屏幕，也不会拿一个性能很高的GPU去搭配1080p屏幕


你也许会在“屏幕分辨率和GPU如何搭配”上一头雾水，不过，好在笔记本电脑是整机卖出的，也就是说，屏幕和GPU是厂家已经给你搭配好来卖给你的，在此方面就不用过多苦恼了。顺带一提，游戏本常用的是2k分辨率

轻薄本/游戏本/全能本

这是笔记本的不同分类

  • 轻薄本注重轻薄便捷，适于日常使用
  • 游戏本注重更高性能，适于游戏使用
  • 全能本介于轻薄本和游戏本之间

操作系统

操作系统是你在使用电脑时用于管理硬件和软件的基础平台，这里给你举两个例子：

  • 国产手机常用Android（安卓）操作系统，当然，各大国内厂商多年来逐渐在Android的基础上进行改进以衍生出了许多不同的操作系统（如鸿蒙OS、Color OS、Origin OS、Realme UI）
  • Apple手机用的操作系统则是iOS

电脑最常见的操作系统是Windows，而Windows中又以win10和win11最为常见，你到时候也大概率将会使用win10或win11作为笔记本电脑的系统

硬盘分区

意为将硬盘划分为多个逻辑部分，假如你的电脑硬盘是一个大仓库，硬盘分区就是将这个大仓库划分为多个独立的小房间，这对电脑来说没什么性能提升，仅方便于用户在不同分区存放不同类型的文件，就像家里不同的房间有不同的用途


在手机的存储空间中通常是没有分区的，电脑则可以通过也必须通过分区来管理文件。系统的默认分区只有"C盘"，可以自己进行分区，分为"D盘"、"E盘"等（C盘、D盘、E盘是分区的名称，后面若还有分区，则按字母顺序类推，不用问为什么没有A盘B盘）

快捷键

快捷键指的是键盘组合键，同时按住键盘上的多个按键来使用一些快速功能，常见如Ctrl + C复制，Ctrl + V粘贴，以下列出常用的快捷键（不用背）：

  • Win + R - 打开运行对话框
  • Win + D - 回到桌面（类似于手机最下方三个快捷按键中的返回主界面）
  • Ctrl + C - 复制
  • Ctrl + V - 粘贴
  • Ctrl + S - 保存
  • Alt + F4 - 强制关闭当前应用
  • Alt + Tab - 在已打开的应用间切换

文件

对电脑上“文件”体系的感知非常重要，是学会使用电脑的必修一课，接下来请确保自己理解“文件实质”、“文件路径”、“文件格式”、“文件的打开方式”以及“文件的层次结构”五个板块

文件实质

你肯定知道“文件”是什么意思，那么为什么还要将这个概念单独拎出来讲呢？在使用手机的过程中，用户虽然也是在与文件打交道，但手机的傻瓜式设计让常年使用手机的用户并不能真正理解到“文件”的实质。那么，文件的实质究竟是什么？这里进行对比讨论便于理解

  • 手机：你拍的照片在「相册」里，下载的音乐在「音乐App」里，系统自动帮你分类好了不同的文件类型，你很少需要主动找文件本身。或许你以为“照片”就是相册里的那张图？其实它本质是一个存储在手机里的图片文件（例如 IMG_001.jpg）
  • 电脑：所有内容，包括文档、图片、视频、软件等，都是以独立文件的形式存在（其实手机也是如此，但手机上你不用找那些独立文件），你需要找到文件本身在哪里。这个时候，你的“照片”是作为一个文件存放在电脑的指定路径内的，需要你自己通过文件路径找到文件在哪里

电脑里的文件就像图书馆的书，每本书（文件）有固定的位置（文件路径），而手机是自动送货上门，你可以直接看到文件内容，而不用管文件放在哪

文件路径

其实手机和电脑都一样，每个文件都有自己的坐标位置，这个坐标位置的表现形式是一条路线，这条路线叫做“文件路径”。不过，在手机和电脑上，用户对“文件路径”的感知并不相同：

  • 手机：手机用户对于“文件路径”的感知几乎没有。例如，“图库”/“相册”这类APP将手机里的所有图片、视频文件搜索出来，并且显示给用户看，因此，用户在使用过程中几乎不需要知道文件实际在哪个位置
  • 电脑：电脑上没有“图库”/“相册”这样的软件，所有文件存放在电脑里，若你需要调用，则要自己找到文件在哪里。以下是一个路径示例：“C:\Users\admin\Laptop.html”，意思是C盘中一个叫Users的文件夹，而Users文件夹中又有一个叫admin的文件夹，这个路径所指向的目标文件就处在admin文件夹中，名字叫“Laptop.html”

文件格式

很容易注意到，此前的叙述中出现了两个文件例名：“IMG_001.jpg”以及“Laptop.html”。在这两个例子中，IMG_001和Laptop都是文件的名字，“.”是一个分隔用的标识符，后面的“jpg”和“html”则是文件的后缀名（又称扩展名），文件扩展名显示了这个文件使用的文件格式


文件格式的作用很简单，它将文件分为不同的类别，一方面告诉电脑，这个文件是做什么用的，应该用什么软件来打开，另一方面也告诉用户，这个文件里面是图片、视频、音频还是文本，又或者是其他的什么内容


文件格式的本质是数据与规则的约定，对人来说，文件格式旨在确保内容被正确地显示出来给人看；对计算机来说，文件格式用来确保数据能用正确的方式解析和处理


以下列举一些常见的文件扩展名：

  • 图片：jpg（等于jpeg），png,gif（动图），ico（图标文件）
  • 视频：mp4，mkv，avi
  • 音频：mp3，wav，flac
  • 压缩包：zip，rar，tar，7z
  • 其他：txt（纯文本），pptx（幻灯片，上课用的ppt就是这个格式），docx（word文档），xlsx（表格，情侣账单就是这个格式），exe（windows系统的程序文件），html（网页文件，你现在看着的这个文件就是使用的html）

特别说明：文件扩展名的作用注定了它不能被随意更改，因此，为照顾小白用户，windows系统初始设置不显示文件扩展名，若要显示文件扩展名需要自己设置。我推荐还是设置一下，只要自己注意重命名文件的时候不要把扩展名（包括那个“.”）删掉，也不要乱改扩展名就可以了

文件的打开方式

我们已经知道，正常情况下每个文件都会带有文件扩展名，不同扩展名对应不同格式，不同格式的文件又需要不同软件来解析并使用，那么，文件总需要一个软件来打开它才能正常使用的


还是以“相册”举例，相册本质上是一个图片文件与视频文件的管理器与播放器。其在文件管理方面发挥的作用在“文件路径”板块已有提及，这里着重说明“播放器”方面，“相册”软件有内置的用于解码图片和视频的解码方式，所以你才能在相册里浏览图片和视频，这也说明了一点：同样可以有其他的软件，带有浏览图片，视频，音频等等文件的功能，选择一个功能全面，效果优良的视频播放器就可以带来更好的观看体验，例如：一个播放器只可以暂停或开始，没有其他功能，另一个播放器却可以调节视频倍速，使用另一个播放器便可以有更好的体验


这个例子中，就是用软件（相册）打开了文件（图片/视频），举一反三，其他类型的文件也可以由不同的软件来打开/播放/更改，也就是说，一个文件可能有多种打开方式


特别说明：只有用于播放图片、视频和音频之类文件的软件或软件内置的一项功能会被称为“播放器”，其他诸如使用浏览器打开html文件，使用压缩软件打开/解压压缩包，这其中的浏览器和压缩软件都不被叫做播放器，“播放器”只是针对于可以播放图片、视频及音频等文件的软件（或软件内置功能模块）的称呼

文件的层次结构

无论是手机还是电脑，其内的文件通常都不是全部堆在一起的。如果沿用上文“硬盘分区”中的例子，将分区比喻为一栋房子中的多个房间，那么每个房间里可能还放置了多个柜子（分区里可能有多个文件夹），柜子可能分为不同区域（文件夹内还可能有多个文件夹），每个区域内还可能放了不同盒子（可能还有第三级文件夹……）（注意我的用词都是“可能”，不代表不可以直接将文件放在硬盘分区内）一个应用软件通常就是在一个根目录下分层存放不同的文件


手机/电脑就是用这种层层嵌套的方式来存放文件的，其道理也很容易想到，便于人自己管理和检索嘛，把文件全堆在一起看得眼花缭乱的，分类一下就好找多了，也非常建议你养成这种分层管理的习惯

快捷方式

刚刚讲到，应用软件通常使用分层管理来存放文件，那如果直接将整个文件夹堆到桌面，就不美观了，而直接将单个文件放在桌面又不能正常使用，这极其的不方便，于是“快捷方式”便诞生了


手机桌面上有许多的应用图标，它们并不是应用本身，而是一个指向应用文件的快捷方式，当你点击这些快捷方式，这些快捷方式会自动指向应用软件实际的文件路径，从而达到既方便又美观的效果


手机通常只能为应用创建快捷方式，而且是自动式的；而电脑可以为任何文件创建快捷方式，包括图片，视频等，电脑在安装应用时通常也会带有“创建快捷方式”的选项，如果勾选这个选项，会自动在桌面上创建该软件的快捷方式

VPN

众所周知，中国有一个国家防火墙（GFW），防火墙会自动屏蔽一些境外网站，使国内用户无法访问诸如google、twitter、youtube等外网


使用VPN则是翻过这道防火墙的方式之一，你在使用VPN时，你的数据会先传输至VPN客户端，VPN客户端将你的数据加密传给VPN服务器，服务器解密后会去访问你想要访问的网址，再将结果返回给你


VPN会隐藏你的真实位置，让防火墙不知道是谁在访问，若你连的VPN是美国节点，网站会以为你是美国用户，由此就可以翻过防火墙访问境外网站


在中国，私自翻墙是违法行为，尤其不能在境外网站发布或传播敏感信息（例如参与政治讨论、发布不当言论等）

模拟器

电脑和手机用的系统不同，两者间的软件自然不互通，若是一个手机游戏并没有PC端，又想要在电脑上玩这款手机游戏，就需要用到模拟器


模拟器是一种通过软件或硬件模仿特定系统、设备或环境行为的工具，使用模拟器可以在某个平台上运行其他平台的软件


通常来说，由于电脑的性能普遍比手机高，所以大多数情况下电脑都可以无压力装一个模拟手机的模拟器，但手机模拟电脑就要求手机有较高的性能。因此，电脑玩手机游戏一般直接下载模拟器即可，但手机想要玩电脑游戏可能就需要用到云游戏

云游戏

云游戏以其占用储存空间小、需求性能低而具有显著的优势，那么相比传统的游戏，云游戏为什么能具有这些优点？


云游戏就是把游戏放在一台远程电脑上运行，你只需要用手机/电脑发出操作并接收画面，发出操作就像是使用遥控器遥控电视，接收画面完全就是在看视频，这也是为什么云游戏的网速要求很高，流量消耗也很高，毕竟你在玩云游戏时相当于一直在看视频


游戏的所有文件都储存在了云游戏服务器上，运行也是在云游戏服务器上运行，自然不需要消耗你的储存空间和性能，云游戏软件本身只作为一个你和服务器互相通信的平台和媒介

软件下载途径
打开edge浏览器（Microsoft Edge），搜索你需要下载的软件名称，会有很多搜索结果，请仔细甄别那些搜索结果是该软件的官网还是某某下载站，注意使用官网进行下载，不要在来路不明的下载站下载软件，如果你不会调整选项，那么里面的捆绑软件会把你的电脑整得乱七八糟
如果这个软件没有官网或者你找不到官网，必须要用某某下载站，也请注意不要点击“安全下载（通过***下载）”，这个安全下载并不安全。一般旁边会有一个普通的下载，使用普通下载即可
软件卸载方式
右键点击你想要卸载的应用的图标（快捷方式），点击“打开文件所在的位置”，此时跳出文件资源管理器，在同级目录里面一般能找到某个名字里带“uninstall”的文件，双击这个文件并进行后续操作即可。如果还是没有这个文件，请直接删除掉整个应用的根文件夹
当然，你也可以选择使用专门的卸载器进行卸载，比如应用列表中的geek
压缩文件/解压压缩包
右键任意文件，在弹出的菜单栏中，你会发现有一项带有 图标的选项“添加到压缩包”，这个选项的功能是将这个文件压缩成压缩包。而若是右键一个压缩包文件，除该选项之外还会多出来三项同样带有该图标的选项，第一项“打开压缩包”是预览压缩包内文件，第二项“提取到当前位置”是将压缩包内的文件直接提取到当前文件夹内，第三项“提取到‘*****\’”是先创建一个与该压缩包同名的文件夹，再将压缩包内的文件提取到这个同名文件夹内（使用该方式需要下载软件7zip）
开机太慢
右键任务栏，点击“任务管理器”，点击“启动”/“启动应用”，里面有很多自启动项，“已启用”表示该应用会在开机时一并打开，这个只需要把每次开机都要用的软件给设置成“已启用”就行，不然设置太多会导致电脑太卡
如果你不认识这个应用，不知道这个应用是干什么的话，不要乱把自启动项关掉
截屏/录屏
截屏：windows自带的最原始的截屏方式就是按下键盘上的PrtSc键，意为PrintScreen，这会将整个屏幕画面复制下来，保存在剪贴板里，可以到其他应用里粘贴使用。除此之外，微信，QQ也附带了参数更为齐全的截屏功能，这里推荐使用应用列表中的Snipaste软件作为截屏软件
录屏：如果是NVIDIA显卡，按下Alt+Z可以打开NVIDIA GeForce Experience，这里内置有录屏功能，直接用这个就行。如果不是N卡用户，可以下载OBS Studio作为录屏工具
